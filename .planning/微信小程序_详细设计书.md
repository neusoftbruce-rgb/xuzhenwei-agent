#### JS 逻辑 (chat.js) 核心流程

```javascript
Page({
  data: {
    mode: 'auto',
    messages: [],        // [{id, role, content, htmlContent, ...}]
    inputText: '',
    loading: false,
    recording: false,
    currentSessionId: null,
    sessions: [],
    drawerOpen: false,
    scrollIntoView: '',
  },

  onLoad() {
    this.checkLogin();
    this.loadSessions();
  },

  // 发送消息 → API → SSE流式接收
  async send() {
    const text = this.data.inputText.trim();
    if (!text || this.data.loading) return;

    // 自动创建会话
    if (!this.data.currentSessionId) {
      await this.createSession(text);
    }

    // 显示用户消息
    this.appendMessage({ role: 'user', content: text });
    this.setData({ inputText: '', loading: true });

    // 调用API
    if (this.data.mode === 'auto') {
      // 智能推荐模式
      const rec = await api.recommend(text);
      this.appendMessage({ role: 'recommend', ...rec });
      this.setData({ loading: false });
    } else {
      // 直接对话模式
      this.streamAIResponse(text, this.data.mode);
    }
  },

  // SSE 流式接收（核心）
  streamAIResponse(message, techniqueId) {
    const that = this;
    const msgId = Date.now();
    this.appendMessage({ id: msgId, role: 'ai', htmlContent: '' });

    wx.request({
      url: `${API_BASE}/api/agent/think`,
      method: 'POST',
      enableChunked: true,  // 关键：启用分块传输
      header: { 'Authorization': `Bearer ${getToken()}` },
      data: {
        message,
        techniqueId: techniqueId || '',
        sessionId: this.data.currentSessionId,
        deepThink: false,
      },
      success: (res) => {
        // 完整响应接收完成
        that.updateMessage(msgId, { loading: false });
      },
    });

    // 监听分块数据（基础库 2.20.1+）
    const requestTask = wx.request({...});
    requestTask.onChunkReceived((chunk) => {
      // 解析 SSE 格式: "data: {...}\n\n"
      const text = that.ab2str(chunk.data);
      const lines = text.split('\n');
      for (const line of lines) {
        if (line.startsWith('data:')) {
          try {
            const event = JSON.parse(line.slice(5).trim());
            if (event.type === 'STEP_CONTENT' && event.content) {
              // 追加内容到当前AI消息
              that.appendToMessage(msgId, event.content);
            }
          } catch(e) {}
        }
      }
    });
  },

  // 语音录制
  startRecord() {
    const recorder = wx.getRecorderManager();
    recorder.start({ format: 'mp3', duration: 60000 });
    this.setData({ recording: true });
  },

  stopRecord() {
    const recorder = wx.getRecorderManager();
    recorder.stop();
    recorder.onStop((res) => {
      // 调用微信语音识别
      this.voiceToText(res.tempFilePath);
    });
    this.setData({ recording: false });
  },

  voiceToText(filePath) {
    wx.showLoading({ title: '识别中...' });
    // 使用微信同声传译插件或后端语音识别
    // 方案A: 微信插件 (免费，但需申请)
    // 方案B: 上传到后端用DeepSeek/Whisper识别
    wx.uploadFile({
      url: `${API_BASE}/api/file/upload`,
      filePath,
      name: 'file',
      success: (res) => {
        const text = JSON.parse(res.data).text;
        this.setData({ inputText: text });
        wx.hideLoading();
      }
    });
  },
});
```

## 5. 小程序申请流程

### 5.1 注册步骤

```
1. 访问 mp.weixin.qq.com
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱、密码
4. 激活邮箱
5. 信息登记:
   - 主体类型: 个人 或 企业
   - 个人: 身份证 + 微信绑定银行卡
   - 企业: 营业执照 + 对公账户打款验证
6. 管理员微信扫码验证
7. 完成 → 获得 AppID
```

### 5.2 审核时间

| 步骤 | 时间 |
|------|------|
| 注册 | 即时（信息提交后） |
| 微信认证（企业） | 1-3 工作日 |
| 首次提交审核 | 1-7 工作日 |
| 后续更新审核 | 几小时 ~ 1天 |

### 5.3 开发阶段（无需审核）

在审核期间可以：
- 用**微信开发者工具**本地开发调试
- 设置为**体验版**，邀请最多 15 人扫码测试
- 用**测试号**（appid: wx...test）立即开始开发

### 6.1 AuthController

```java
@RestController
@RequestMapping("/api/auth")
public class AuthController {

    @Value("${wechat.appid}")
    private String appId;

    @Value("${wechat.secret}")
    private String secret;

    @PostMapping("/wechat-login")
    public Map<String, Object> wechatLogin(@RequestBody Map<String, String> body) {
        String code = body.get("code");
        // 1. 调微信接口 code → openid
        String url = "https://api.weixin.qq.com/sns/jscode2session" +
            "?appid=" + appId + "&secret=" + secret + "&js_code=" + code +
            "&grant_type=authorization_code";
        // 2. 查/建用户
        // 3. 生成JWT
        // 4. 返回token
    }
}
```

### 6.2 JWT Filter

```java
@Component
public class JwtAuthFilter extends OncePerRequestFilter {
    @Override
    protected void doFilterInternal(HttpServletRequest request,
            HttpServletResponse response, FilterChain chain) {
        String header = request.getHeader("Authorization");
        if (header != null && header.startsWith("Bearer ")) {
            String token = header.substring(7);
            // 验证JWT → 设置 SecurityContext
        }
        chain.doFilter(request, response);
    }
}
```
